ยินดีต้อนรับเข้าสู่เว็บไซค์เพื่อการศึกษาสำหรับการเรียนการสอน วิชาพื้นฐานการสร้างเว็บไซค์
คำอธิบายรายวิชา ศึกษาและปฎิบัติเกี่ยวกับโครงสร้างของเว็บไซค์และสร้างเว็บไซค์ด้วยโปรแกรมภาษ HTML5 การตกแต่งข้อความ การแทรกรูปภาพหรือสื่อต่างๆ การสร้างตาราง การสร้าง การเชื่อมโยงในแบบต่างๆ การสร้างเฟรม การออกแบบและการสร้างฟอร์ม การแทรก Java Script การทดสอบการทำงานของเว็บไซค์ไปยังเว็บเซิร์ฟเวอร์
